# Imágenes para el README

Para completar el README de OpenBookDrive, agrega las siguientes imágenes en esta carpeta:

## Imágenes requeridas

| Archivo | Descripción | Tamaño recomendado |
|---------|-------------|-------------------|
| `banner.png` | Banner principal del proyecto | 1200x400 px |
| `screenshot.png` | Captura de pantalla principal | 900x600 px |
| `demo.gif` | Demo animado de la aplicación | 700x400 px |

## Cómo obtenerlas

### Banner
- Crea uno gratis en [Canva](https://canva.com) buscando "GitHub README banner"
- O usa [carbon.now.sh](https://carbon.now.sh) para código

### Screenshots
- Toma una captura de tu aplicación en [openbookdrive.vercel.app](https://openbookdrive.vercel.app)
- Herramientas: Lightshot (Windows), Cmd+Shift+4 (Mac), Flameshot (Linux)

### Demo GIF
- [ScreenToGif](https://www.screentogif.com/) — Gratis, Windows
- [GIPHY Capture](https://giphy.com/apps/giphycapture) — Mac/Windows
- [LICEcap](https://www.cockos.com/licecap/) — Gratis, simple

## Placeholders

Si no tienes imágenes aún, el README usará placeholders de via.placeholder.com automáticamente.
